# Q-Manager

React를 이용해 개발한 자격증 관리 웹 프로젝트입니다.

사용자가 자격증을 검색하고 등록할 수 있으며, 시험 일정과 D-Day를 확인하고 즐겨찾기 기능을 이용해 관심 있는 자격증을 관리할 수 있습니다.

1. 개발 환경

- React
- Vite
- JavaScript
- React Router
- LocalStorage
- Q-Net Open API

2. 주요 기능

- 로그인 / 회원가입
- 자격증 검색
- 자격증 등록, 수정, 삭제
- 즐겨찾기
- 시험 일정 조회
- D-Day 계산
- 마이페이지
- FAQ

3. 프로젝트 구조

```text
src
├── components
├── loginBox
├── qualificationsData
├── utils
├── App.jsx
└── main.jsx
```

4. 실행 방법

```bash
npm install
npm run dev
```

5. 개선 사항

- 백엔드 및 데이터베이스 연동
- 사용자별 데이터 관리
- 반응형 UI 적용
- 로그인 기능 개선
